Projets GSB

Présentation des projets Back-GSB et Front-GSB.

Back-GSB

Back-GSB est une application back-end pour la gestion de médecins. Il met à disposition des api REST permettant d'obtenir et de modifier une liste de médecins triés par département ou non. La modification de la liste des médecins est protégée par un JWT (Json Web Token). Donc seule une personne autorisée peut la modifier.

fonctionnalités principales de Back-GSB :

  • Gestion des médecins : lecture, ajout, modification et suppression des médecins
  • Liste des médecins par département
  • Protection des modifications des medecins avec un JWT

Technologies utilisées :

  • IDE : Visual Studio Community 2019
  • Langage de programmation : C#
  • Framework : .NET
  • SGBD : Microsoft SQL Server Management Studio
  • Modélisation de la base de données : Looping
  • Outil de contrôle de version : Git (avec dépôt sur GitHub)

Front-GSB

Front-GSB est une application front-end qui interagit avec l'API Back-GSB pour afficher et gérer les informations sur les médecins. Elle permet aux visiteurs médicaux du laboratoire Galaxy Swiss Bourdin (GSB) de consulter et trier les médecins par département, ainsi que d'effectuer des actions spécifiques sur les médecins.

Les fonctionnalités principales de Front-GSB :

  • Interface pour la gestion des médecins : lecture, ajout, modification et suppression des médecins
  • Interface pour la liste des médecins par département
  • Protection des modifications des médecins avec une demande d'authentification

Technologies utilisées :

  • IDE : Visual Studio Community 2019
  • Langage de programmation : C#
  • Framework : .NET
  • API : Back-GSB
  • Outil de contrôle de version : Git (avec dépôt sur GitHub)
